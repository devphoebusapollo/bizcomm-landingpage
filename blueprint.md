# Project Blueprint: BizComm Landing Page

## Overview

This document outlines the structure, features, and development plan for the BizComm landing page, a single-page website designed to promote a business communication course.

## Implemented Features

### Core Structure

*   **HTML5:** The page is built with semantic HTML5 for accessibility and SEO.
*   **CSS3:** Styling is done with modern CSS, including Flexbox and Grid for layout.
*   **JavaScript (ES6+):** The page uses modern JavaScript for interactive features.

### Interactive Components

*   **Responsive Navigation:** A hamburger menu provides a mobile-friendly navigation experience.
*   **FAQ Accordion:** The frequently asked questions section uses an accordion to show and hide answers.
*   **Enrollment Modal:** A modal window appears when the user clicks the "Enroll Now" button.
*   **Carousels (Web Component):** The testimonial and curriculum carousels have been refactored into a reusable, modern Web Component with a polished design and responsive behavior.
*   **Animated Gallery:** An image gallery with a staggered animation effect that triggers on scroll.
*   **Smooth Scrolling:** The navigation links smoothly scroll to the corresponding sections of the page.

### Design and Style

*   **Typography:** The design uses the "Poppins" font from Google Fonts.
*   **Icons:** Font Awesome is used for icons throughout the page.
*   **Imagery:** The page uses high-quality images from Pexels to create a professional look and feel.
*   **CSS Variables:** The stylesheet has been refactored to use CSS variables for colors and fonts, making the design easier to maintain and update.
